# ✈️ Airline Performance Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing airline operational performance and passenger transportation trends using SQL, Excel, Power BI, and Tableau. The dashboard provides actionable insights into airline efficiency, passenger traffic, route performance, and load factor metrics to support data-driven decision-making.

---

## 🎯 Objective

The primary objective of this project is to:

- Analyze airline operational data.
- Monitor passenger transportation trends.
- Evaluate airline performance across routes.
- Identify key business insights through KPI reporting.
- Develop interactive dashboards for effective visualization.

---

## 📊 Dataset Information

The dataset contains airline operational records, including:

- Passenger Count
- Flight Operations
- Route Information
- Airline Details
- Load Factor Metrics
- Transportation Trends

Total Records Analyzed: **10,000+**

### 2. Data Cleaning

- Handled missing values.

### 3. Data Analysis
- Performed SQL queries for KPI generation.
- Analyzed passenger traffic trends.

### 4. Dashboard Development
- Created Power BI dashboards.
- Developed Tableau visualizations.
- Implemented dynamic filtering and drill-down functionality.
